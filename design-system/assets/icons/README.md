# Icons

This folder contains a small SVG icon set for UA Help Context promotional graphics and Claude Design usage.

## Included files

- `svg/context-bubble.svg` - context, explanation, and helpful information.
- `svg/guidance-path.svg` - practical next steps and direction.
- `svg/help-hand.svg` - support and help.
- `svg/search-context.svg` - finding relevant context.
- `svg/shield-check.svg` - trust and safety.
- `svg/steps-list.svg` - clear process and action checklist.
- `icons.json` - structured metadata for the icon set.

## Style

- Simple line icons
- 24px viewBox
- Rounded caps and joins
- Current-color stroke for easy recoloring
- Minimal detail for high readability at small sizes

Use Ukrainian blue `#2563EB`, yellow `#FACC15`, or neutral text `#111827` when applying icons in Claude Design.
