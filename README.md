# ua.help-context.com

UA Help Context is a Ukrainian informational and promotional web project focused on clear context, practical guidance, and useful next steps.

## Design system

The Claude Design brand kit lives in [`design-system/`](design-system/).

Use this folder as the source for promotional graphics, brand direction, color tokens, typography guidance, and future visual assets for `ua.help-context.com`.

Current brand kit files:

- [`design-system/README.md`](design-system/README.md) - usage notes for Claude Design.
- [`design-system/brand.md`](design-system/brand.md) - human-readable brand direction.
- [`design-system/brand.json`](design-system/brand.json) - structured brand data.
- [`design-system/tokens.css`](design-system/tokens.css) - CSS variables for colors, typography, spacing, and radius.
- [`design-system/assets/`](design-system/assets/) - folders for logos, fonts, images, and icons.
